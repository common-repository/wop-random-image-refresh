== Description ==
The WOP Random Image Refresh plugin help you to show a different image every time the page loads. The plugin very easy to use and also provide feature to change text/ image caption color according to pictures. The plug in uses short codes to make the frontend feature new images.

The plug in can be used to refresh any image such as the hero banner image or any other on the page even multiple images on a page. The plug in works well for posts, pages, custom post, widgets and sidebars.

== Requirements ==
This plugin requires PHP 5 or PHP 7 and WordPress 3.0 or greater. It works in standard WordPress environments. It is strongly recommended to use WordPress in the newest version.

== Installation ==

1. Go to Plugins > Add New. 
2. Searh "wop random image refresh"
3. install and activate the WordPress Plugin.

== How to Use ==

1. Insert shortcode "[wop_random_image_refresh]" on page or post where you want to have an image refresh.
2. Insert attachment ID into shortcode "attachment="12, 11"" Example: "[wop_random_image_refresh attachment="12, 11"]". You can get attachment idea from media library (pleaes refer our screenshot.)
3. For Caption: Insert "caption="true" Example: "[wop_random_image_refresh attachment="12, 11" caption="true"]".
4. For Caption text color according to image insert "color="true":  Example: "[wop_random_image_refresh attachment="12, 11" caption="true" color="true"]".
